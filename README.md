# 📖 Overview

**Nova City** is a next-generation interactive portfolio that reimagines how personal websites can be experienced.

Instead of navigating static pages, visitors embark on a cinematic journey through space, descend onto humanity's first futuristic city, explore intelligent districts, and discover portfolio content inside immersive holographic buildings.

The project combines **3D graphics**, **procedural environments**, **cinematic storytelling**, and **modern frontend engineering** into a seamless interactive experience.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/NovaCity.git
```

Navigate into the project

```bash
cd NovaCity
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

# 💡 Design Philosophy

Nova City was created around one simple idea:

> **A portfolio should be explored—not merely viewed.**

Instead of presenting information through conventional pages, Nova City transforms projects, skills, education, and experience into destinations inside an immersive digital world, creating a memorable experience for recruiters, collaborators, and developers.
